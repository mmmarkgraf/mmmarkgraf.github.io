mRNA Reconstruction from NGS RNA-Seq

Project Date November, 2020
Techniques Bioinformatics, Next-Generation Sequencing, Genetics, Genetic Alignment, Big Data
Visit GitHub Repository
Medium Article

The goal of this project was to reconstruct mRNA from RNA-Seq data. In this specific situation, the RNA-Seq was extracted from healthy patients and patients diagnosed with Type 2 Diabetes (T2D). Thus, this project examines the preproinsulin mRNA (pre-precursor of the insulin hormone/protein) of pancreatic beta cells.

The analytical results from this study did not show any distinguishable variations between either patient types, concluding that the cause of T2D in the four patients of this study did not result from the insulin gene but other factors.
